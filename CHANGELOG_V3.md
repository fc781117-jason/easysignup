# 消防局多功能報名系統 V3 更新紀錄

## 版本識別
- 登入頁副標題顯示：`V3.0 欄位與案件管理優化版`
- 側邊欄顯示：`報名系統 V3`
- `styles.css`、`app.js`、`firebase-config.js` 已加入 `?v=3.0.0`，降低瀏覽器快取造成看不到更新的機率。

## 本版修正
1. 移除側邊欄「新增 / 編輯案件」，功能合併至「案件管理」。
2. 案件管理頁保留「＋ 新增案件」，單一案件後續處置由「案件管理」按鈕進入。
3. 單一案件管理內提供：編輯本案件設定、切換開放 / 截止、刪除本案件、Excel 名冊、PDF 簽到表、PDF 餐食簽收表、PDF 統計報告。
4. 最高系統管理員可管理全部案件；一般承辦人只能管理自己建立的案件。
5. 一般填寫者在案件列表只看到「填寫」與「重新編輯」；重新編輯會更新原資料，序號不變。
6. 欄位設定文案改為：提供固定模板欄位，如承辦人有其他需求，可自行點選取消或新增擴充。
7. 固定欄位支援啟用 / 停用：粉紅色為啟用，灰色為停用。
8. 有選項的欄位可點 `▾` 展開，刪除既有選項或新增選項。
9. 自訂欄位支援新增、刪除、啟用 / 停用、欄位名稱、欄位類型、必填與選項編輯。

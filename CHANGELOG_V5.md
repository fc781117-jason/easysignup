# V5 PDF 版面與統計視覺優化

## 修正重點

1. 簽到表與餐食簽收表改為橫式 A4，避免簽名欄過窄。
2. 重新配置欄寬：序號縮小、單位依內容保留適當寬度、簽到／簽退／簽收欄位加寬。
3. 簽名欄列高加高，讓紙本簽到、簽退、簽收更實用。
4. 統計報告改為視覺化版面，加入摘要卡、圓餅圖、長條圖與文字說明。
5. 統計報告支援性別、餐食、停車、年齡、單位、職稱等項目視覺化呈現。
6. PDF 產生函式支援 portrait / landscape 兩種方向。

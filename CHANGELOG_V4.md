# 消防局多功能報名系統 V4 更新紀錄

## V4.0 PDF 輸出修正版

本版重點修正 PDF 無法產出的問題，並保留 V3 的欄位設定、案件管理、權限分類與輸出邏輯。

### 已修正

- 修正 `pdfCanvas` 原本套用 `hidden` 導致 `html2canvas` 無法正確截圖的問題。
- PDF 產出時會暫時顯示於不可見畫布，再轉成 PDF，完成後自動隱藏。
- 手機 Safari / iOS 環境若無法直接下載，會改以預覽或開啟 Blob PDF 的方式處理。
- 若 jsPDF 或 html2canvas 套件載入失敗，系統會自動開啟「列印／另存 PDF」預覽頁作為備援。
- PDF 按鈕文字調整為「預覽／下載 PDF」，更符合手機與桌機不同下載行為。
- PDF 報名表、簽到表、餐食簽收表、統計報告皆套用同一組穩定輸出流程。

### 更新辨識

- 登入頁副標題顯示：`V4.0 PDF 輸出修正版`
- 側邊欄顯示：`報名系統 V4`
- `styles.css`、`app.js`、`firebase-config.js` 已加入 `?v=4.0.0`，降低瀏覽器快取造成看不到更新的機率。
